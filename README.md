# HeoMusic

![](/img/cover.webp)

一个基于Aplayer和Meting的静态音乐播放器

## 修改歌单

请根据[MetingJS规则](https://github.com/metowolf/MetingJS)编辑`index.html`的Meting-js相关代码

![修改id和server](/img/help.webp)

## 预览

[HeoMusic - 用音乐感染人心](https://music.zhheo.com/)

## 基本操作

`Space空格键`：暂停/播放音乐
`上/下方向键`：增加/减少音量
`左/右方向键`：上一曲/下一曲
